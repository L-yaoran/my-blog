    一、单选题（10 题）
    第 1 题单选题
    计算机系统通常由（ ）组成。
    A 硬件资源和软件资源
    B 鼠标和键盘
    C 浏览器和数据库
    D 微信和 PyCharm
    ✓ 正确
    解析：计算机系统由硬件资源（CPU、内存、硬盘等）和软件资源（操作系统、应用软件等）组成。
    第 2 题单选题
    操作系统的主要作用是（ ）
    A 只负责画图
    B 管理硬件和软件资源
    C 只负责聊天
    D 只负责写 Python 代码
    ✓ 正确
    解析：操作系统负责管理计算机的硬件和软件资源，让应用软件能够使用 CPU、内存、硬盘、网络等资源。
    第 3 题单选题
    Linux 在真实工作中最常见的使用场景是（ ）
    A 服务器
    B 游戏手柄
    C 纸质文档
    D 鼠标垫
    ✓ 正确
    解析：Linux 在服务器领域占据主导地位，大多数互联网公司的后端服务都运行在 Linux 服务器上。
    第 4 题单选题
    Ubuntu 和 CentOS 都属于（ ）
    A Linux 发行版
    B Python 库
    C 浏览器插件
    D 数据库表
    ✓ 正确
    解析：Ubuntu 和 CentOS 都是 Linux 的发行版，基于 Linux 内核，搭配不同的软件包和桌面环境。
    第 5 题单选题
    SSH 默认端口通常是（ ）
    A 21
    B 22
    C 80
    D 3306
    ✓ 正确
    解析：SSH 默认使用 22 端口。21 是 FTP，80 是 HTTP，3306 是 MySQL。
    第 6 题单选题
    Linux 的根目录是（ ）
    A C:\
    B D:\
    C /
    D root:
    ✓ 正确
    解析：Linux 只有一个根目录 /，所有文件和目录都在根目录下。Windows 有多个盘符如 C:\、D:\。
    第 7 题单选题
    查看当前所在目录的命令是（ ）
    A pwd
    B mkdir
    C touch
    D rm
    ✓ 正确
    解析：pwd（Print Working Directory）显示当前所在目录的完整路径。
    第 8 题单选题
    创建多级目录时，常用的命令是（ ）
    A mkdir -p a/b/c
    B touch -p a/b/c
    C cat -p a/b/c
    D clear -p a/b/c
    ✓ 正确
    解析：mkdir -p 会在父目录不存在时自动创建，可以一次性创建多级目录。
    第 9 题单选题
    > 和 >> 的区别是（ ）
    A > 追加，>> 覆盖
    B > 覆盖，>> 追加
    C 两者完全一样
    D 两者都只能删除文件
    ✓ 正确
    解析：> 覆盖写入（替换原内容），>> 追加写入（保留原内容，在末尾添加）。
    第 10 题单选题
    vim 中按 i 的作用是（ ）
    A 进入输入模式
    B 直接退出
    C 删除文件
    D 打包文件
    ✓ 正确
    解析：在 vim 的命令模式下按 i 进入输入模式（Insert mode），可以开始编辑文本。

    ✓ 正确
    解析：Linux 使用 / 作为路径分隔符，如 /home/user/file.txt。Windows 使用 \。

    ✓ 正确

    第 18 题填空题
    touch file.txt 的作用是创建一个。
    ✓ 正确
    解析：touch 命令创建空文件，如果文件已存在则更新修改时间。
    第 19 题填空题
    rm 删除文件后通常不会进入。
    ✓ 正确
    解析：Linux 的 rm 命令直接删除文件，不经过回收站，删错很难恢复。
    第 20 题填空题
    vim 中 :wq 表示。
    ✓ 正确
    解析：:w 表示保存（write），:q 表示退出（quit），:wq 表示保存并退出。

    ✓ 正确
    参考答案：因为虚拟机安装耗时较长，且不同电脑可能遇到配置问题；云服务器创建更快，也更接近真实工作中的服务器使用方式。

    第 30 题简答题
    vim 的三种模式分别是什么？如何保存退出？
    查看参考答案 ▼
    vim 有命令模式、输入模式、底线命令模式。打开文件后按 i 进入输入模式，编辑完按 Esc 回到命令模式，再输入 :wq 保存退出。
    ✓ 正确
    参考答案：vim 有命令模式、输入模式、底线命令模式。打开文件后按 i 进入输入模式，编辑完按 Esc 回到命令模式，再输入 :wq 保存退出。
