    一、单选题（10 题）
    第 1 题单选题
    闭包的三大构成条件是（ ）。
    A 有循环、有判断、有返回
    B 有嵌套、有引用、有返回
    C 有参数、有注释、有调用
    D 有类、有对象、有继承
    ✓ 正确
    解析：闭包三条件：有嵌套（外部函数包含内部函数）、有引用（内部函数引用外部变量）、有返回（外部函数返回内部函数名）。
    第 2 题单选题
    下列属于闭包语法核心的是（ ）。
    A 外部函数返回内部函数名
    B 内部函数必须无参
    C 外部函数必须是全局函数
    D 内部函数必须使用 global
    ✓ 正确
    解析：闭包的核心是外部函数返回内部函数名（函数引用），这样外部才能调用内部函数并保留闭包环境。
    第 3 题单选题
    闭包最典型的作用是（ ）。
    A 提高网络速度
    B 保存外层变量并延长其生命周期
    C 自动生成类
    D 替代 if 判断
    ✓ 正确
    解析：闭包让内部函数持有外层变量的引用，即使外层函数执行完毕，变量仍不会销毁，延长了其生命周期。
    第 4 题单选题

    A global
    B nonlocal
    C static
    D lambda
    ✓ 正确
    解析：nonlocal 用于在内部函数中声明要修改的外层（非全局）变量，global 用于全局变量。

    第 33 题代码实战
    挑战题：函数耗时统计装饰器
    实现一个"函数耗时统计装饰器"，输出函数名和执行时长。
    提示：使用 time.time() 记录开始和结束时间，计算差值。
    查看参考答案 ▼ 
    import time

def timer(func):
    def inner(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"函数 {func.__name__} 执行耗时: {end - start:.4f} 秒")
        return result
    return inner

@timer
def slow_func():
    total = 0
    for i in range(1000000):
        total += i
    return total

print(slow_func())
    ✓ 正确
    参考答案：
    import time

def timer(func):
    def inner(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"函数 {func.__name__} 执行耗时: {end - start:.4f} 秒")
        return result
    return inner

@timer
def slow_func():
    total = 0
    for i in range(1000000):
        total += i
    return total

print(slow_func())

    提示：用一个全局变量模拟登录状态，装饰器内部判断状态后决定是否调用原函数。

    # 模拟登录状态
is_login = False

def login_required(func):
    def inner(*args, **kwargs):
        if is_login:
            return func(*args, **kwargs)
        else:
            print("请先登录后再操作！")
    return inner

@login_required
def view_secret():
    print("这是机密内容...")

# 未登录调用
view_secret()  # 输出: 请先登录后再操作！

# 登录后调用
is_login = True
view_secret()  # 输出: 这是机密内容...
